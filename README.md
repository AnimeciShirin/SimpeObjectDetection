# Building
Install libraries on below<br/>
```
pip install pytorch
```
```
pip install pillow
```
```
pip install timm
```
<br/>

To download dataset automatically, run the script first time. It won't download again next time.<br/>
```
python main.py
```

# Usage
<ins>Run the terminal as Administrator.</ins><br/>

```
python main.py
```
